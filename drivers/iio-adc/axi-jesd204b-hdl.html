Description

The AXI JESD204 HDL driver is the driver for the HDL interface core which is used on various FPGA designs interfacing to the AD9250. The driver is implemented as an Open Firmware Device Tree (DT) platform driver. It’s register map can be found here: Register access

Example platform device initialization

The AXI JESD204B driver is a platform driver and can currently only be instantiated via device tree.

Required devicetree properties:

  • compatible: Should always be *xlnx,axi-jesd204b-rx2-1.00.a* or *xlnx,axi-jesd204b-rx4-1.00.a*

  • reg: Base address and register area size. This parameter expects a register range.

  • clocks: Clock provider phandle

  • clock-name: Clock input name string

  • jesd,frames-per-multiframe: Number of frames per multi-frame

  • jesd,bytes-per-frame: Number of bytes (octets) per frame

  • jesd,lanesync_en: Enable lane synchronization

  • jesd,scramble_en: Enable scrambling

Example:

axi_jesd204b_rx2_0: axi-jesd204b-rx2@77a00000 {
    compatible = "xlnx,axi-jesd204b-rx2-1.00.a";
    reg = < 0x77a00000 0x10000 >;
    clocks = <&ad9250_clkin>;
    clock-names = "clkin";
    jesd,lanesync_en;
    jesd,scramble_en;
    jesd,frames-per-multiframe = <32>;
    jesd,bytes-per-frame = <2>;
    xlnx,cf-buftype = <0x0>;
    xlnx,dphase-timeout = <0x8>;
    xlnx,family = "kintex7";
    xlnx,num-mem = <0x1>;
    xlnx,num-reg = <0x1>;
    xlnx,s-axi-min-size = <0x1ff>;
    xlnx,slv-awidth = <0x20>;
    xlnx,slv-dwidth = <0x20>;
    xlnx,use-wstrb = <0x0>;
} ;
